# LBO Model — TechRetail SA
# Entry assumptions
company_name = "TechRetail SA"
entry_ev = 450_000_000  # EUR 450M
entry_ebitda = 60_000_000  # EUR 60M at 7.5x
entry_multiple = 7.5
revenue_y0 = 300_000_000
ebitda_margin_y0 = 0.20
revenue_cagr = 0.08
margin_expansion = 0.01  # +1% per year
capex_pct = 0.03
wc_change_pct = 0.01
da_pct = 0.04
tax_rate = 0.28
holding_years = 5

# Debt structure
senior_debt = 247_500_000  # 55% of EV at 6%
senior_rate = 0.06
mezz_debt = 67_500_000   # 15% of EV at 10%
mezz_rate = 0.10
equity = 135_000_000      # 30% of EV

Build year-by-year projections for 5 years:
- Income statement: Revenue, EBITDA, DA, EBIT, Senior Interest, Mezz Interest, EBT, Tax, Net Income
- Debt schedule: Opening Senior, Mandatory repayment 5% pa, Closing Senior; Mezzanine PIK (no cash interest, accrues to principal)
- Cash flow: EBITDA - Capex - WC change - Cash Interest - Tax = Free Cash Flow to Debt
- Debt paydown: use FCF to repay senior first, then mezz

Exit analysis (3 scenarios):
- Bear case: 6.0x exit multiple
- Base case: 7.5x exit multiple  
- Bull case: 9.0x exit multiple

For each scenario calculate:
- Exit EV = Exit multiple x Year 5 EBITDA
- Total debt at exit (remaining)
- Equity value = Exit EV - Total Debt
- Equity MOIC = Exit Equity / Entry Equity
- IRR = (Exit Equity / Entry Equity)^(1/5) - 1
- DPI = Exit Equity / Entry Equity (same as MOIC for single investment)

Print all tables using tabulate with headers.
Save to lbo_output.xlsx with openpyxl: 3 sheets (Income Statement, Debt Schedule, Returns Summary).

# VaR Risk Dashboard
Portfolio Value-at-Risk calculator implementing Historical, Parametric, and Monte Carlo methods.

- Monte Carlo VaR: Cholesky decomposition for correlated asset simulation (10,000 paths)
